# harness-learn

实战学习仓库，专注于 **Harness**、**SDD** 和 **Open Code** 三大主题。

## 主题

| 主题 | 说明 |
|------|------|
| **Harness** | Harness CD/CD 平台实践，包括 Pipeline、Feature Flags、Cloud Costs 等模块 |
| **SDD** | 理解并实践 Service-Driven Development 理念 |
| **Open Code** | 探索 Open Code 相关工具链与工作流 |

## 目录结构

```
├── docs/           # 学习笔记与文档
├── examples/       # 示例代码与配置
└── projects/       # 实战项目
```

> 目录结构会随学习进度逐步完善。

## 目标

- 深入理解 Harness 平台的核心概念与最佳实践
- 掌握 SDD 方法论并应用到实际项目中
- 熟悉 Open Code 生态，提升开发效率
